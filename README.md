# ClusterExhibit
## Virtual Exhibition Platform (K-Means + Streamlit)

ML-based virtual exhibition system built using K-Means Clustering.

Automatically groups similar exhibition items into meaningful clusters.

Interactive interface created with Streamlit for easy exploration.

Users can upload data and instantly view cluster assignments & visuals.

Fully deployed using Streamlit Cloud for online accessibility.
